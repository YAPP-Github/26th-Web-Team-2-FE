| 타입 | 설명 |
| --- | --- |
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `docs` | 문서 수정 (README 등) |
| `style` | 코드 스타일 수정 (세미콜론, 공백 등) |

| `chore` | 빌드, 설정, 패키지 등 기타 작업 |
| `perf` | 성능 개선 |
| `ci` | CI 관련 변경 |
